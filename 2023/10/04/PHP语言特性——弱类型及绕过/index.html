PHP弱类型及其绕过

yu1e
2023-10-04
WebSecurity

什么是弱类型？

弱类型是可以随意转换变量的类型，PHP不会验证变量的类型，可以随时的转换类型

Hash比较缺陷

PHP在处理hash字符串的时候，如果通过!=或者==来对hash值进行比较，这会把每一个以0e开头的hash值都解释为0，所以如果两个不同的密码经过hash以后，其hash值都是以0e开头的，那么PHP会认为他们相同，都是0

extract()函数介绍

extract()函数接收一个键值对数组作为参数，返回一个符号表，将键名作为变量名，键值作为变量值。

ereg正则%00截断及strpos、ereg用数组返回NULL

ereg()函数，int ereg(string pattern, string string, array [regs])，以 pattern 的规则来解析比对字符串 string。比对结果返回的值放在数组参数 regs 之中，regs[0] 内容就是原字符串 string、regs[1] 为第一个合乎规则的字符串、regs[2] 就是第二个合乎规则的字符串，余类推。若省略参数 regs，则只是单纯地比对，找到则返回值为 true。 搜索对于字母字符是区分大小写的，用于正则表达式匹配。

strpos() 函数，strpos(string,find,start)，string指定要搜索的字符串，find指定要查找的字符串，start可选，规定在何处开始查找，返回find在字符串string中第一次出现的位置。

  • ereg()函数存在NULL截断漏洞,可以%00截断，遇到%00则默认为字符串的结束，所以可以绕过一些正则表达式的检查。
  • ereg()只能处理字符串的，遇到数组做参数返回NULL。
  • 空字符串的类型是string，NULL的类型是NULL,false、true是boolean类型
  • strpos()函数如果传入数组，便会返回NULL
  • ereg()函数在PHP7.0+之后被删除

绕过escapeshellarg+escapeshellcmd函数

原理

escapeshellarg()函数，将给字符串增加一个单引号并且能引用或者转码任何已经存在的单引号，这样以确保能够直接将一个字符串传入 shell 函数，并且还是确保安全的。

escapeshellcmd()函数，对字符串中可能会欺骗 shell 命令执行任意命令的字符进行转义。反斜线（\）会在以下字符之前插入： &#;`|?~<>^()[]{}$, \x0A 和 \xFF。 *’ 和 “ 仅在不配对儿的时候被转义。

这里的代码都是在Linux下运行的，Windows下运行结果不一样。

两个函数都会对单引号进行处理，但是是有区别的

对于单个单引号, escapeshellarg() 函数转义后,还会在左右各加一个单引号,但 escapeshellcmd() 函数是直接加一个转义符

对于成对的单引号, escapeshellcmd() 函数默认不转义,但 escapeshellarg() 函数转义

这里简单说一下这个成对的单引号是怎么看的，也就是escapeshellcmd()怎么界定成对的概念，即从第一个单引号开始，每两个是一对，最后单出来的那个作为单个的单引号进行转义
